# Design loader (reference only)

This folder maps React routes to the **original** HTML designs in `/design`. Do not edit files under `/design`.

| React route | Design source | Notes |
|-------------|---------------|--------|
| `/` (Home) | `../design/landing_page/code.html` | Landing / homepage (`Home` alias) |

Use `pages.json` and `index.ts` for programmatic lookups when converting additional pages.
